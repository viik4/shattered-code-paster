# Shattered Code Paster

A small desktop tool for the Fortnite **Shattered** fragment hunt on fortnite.gg. Paste your list of codes, hit Start, and it enters them one at a time so you don't have to type all 1296 by hand.

Made for the community to help people who are just getting started.

## Features

- Paste your codes, or load them from a `.txt` (one per line)
- Clears the field and enters each code automatically
- Optional Enter-to-submit after each code
- Adjustable speed and a start countdown
- Start and Stop buttons, plus Esc to stop anytime
- Themed orange-and-blue interface to match the event

## Requirements

- Windows
- The prebuilt `.exe` needs nothing installed. 

## How to use

1. Open the https://fortnite.gg/shattered and sign in.
2. Run the app and paste in your codes.
3. Press Start, then click the site's code box during the countdown.
4. It enters each code wherever your cursor is focused.

## Notes

- It only simulates keystrokes into whatever field is focused. It does not connect to any website or touch your account.
- It cannot get past a CAPTCHA. If entries stop landing, the site is asking for a human, so do that step yourself and press Start again.
- Bump the delay up if codes get skipped. That usually means the site needs a moment between entries.
- Windows may flag a fresh PyInstaller build through SmartScreen. That is normal for tools that simulate input. Allow it through if you trust the source.
